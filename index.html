Vulkan Renderer

C++ · Vulkan

This project tied as my favorite with the NES project. It was part of the "Graphics Programming 2" course and is a real time 3D renderer written with the Vulkan API. The project uses the following rendering techniques: Depth Buffering, Deferred Rendering with Depth Prepass, Physically Based Rendering using Image-Based Lighting with HDR images, Cook-Torrance BRDF materials (using textures), Post Processing chain with Tone Mapping and it supports point and directional lights.
